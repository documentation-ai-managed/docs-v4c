## Overview

To start using V4c:

1. Sign up at `https://v4c.io/register`
2. Activate your account via email
3. Log in to the dashboard

## First Steps

- Create your first project doc space
- Invite collaborators via their email
- Organize documentation with folders

For more guidance, check our quick start guide or contact support.
