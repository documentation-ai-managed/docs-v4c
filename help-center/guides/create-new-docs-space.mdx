## Overview

Start organizing your project docs in a few steps.

<Steps>
  <Step title="Open dashboard">
    Go to your V4c dashboard after logging in.
  </Step>
  <Step title="Click 'New Space'">
    Press the "New Space" button at the top right.
  </Step>
  <Step title="Fill details">
    Enter a name and description for your documentation space.
  </Step>
  <Step title="Save and invite team">
    Click save, then invite team members from the Share tab.
  </Step>
</Steps>
